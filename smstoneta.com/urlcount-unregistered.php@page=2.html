Please note Important -
- This page shows URL counts along with voter IDs for URLs created by registered users via sending to 08141277555 a sms containing short URLs created using these sites - http://tny.cz/ (In these sites, URL changes with each edit of content).

- Valid facebook URLs having any of these formats not having "?" in the URL will also be accepted -
http://facebook.com/USERNAME/posts/PURE_NUMBER , http://fb.com/USERNAME/posts/PURE_NUMBER , http://www.facebook.com/USERNAME/posts/PURE_NUMBER , http://www.fb.com/USERNAME/posts/PURE_NUMBER
(example - https://www.facebook.com/mehtarahulc/posts/10152679852011922)

- Admin is not responsible in anyway for the content of the URLs displayed in this page. Reader discretion is advised before giving any opinion on the URLs like checking for edit history of the FB links.

- A new URL can be created only from a registered mobile. A newly created URL will be partially hidden. Full URL will be shown only if 50 or more registered or activated mobiles send SMS-es to support it.

- After activation or registration of your mobile, you can open each URL and decide if you would like to support or oppose it. Numerical codes for supporting, opposing a URL or cancelling your opinion is given in the table below.
- Alphabetic Codes for supporting a URL is the URL itself to be sent to our server number, 08141277555 via sms.
- Alphabetic oppose SMS-code for a registered URL is the URL followed by *n , alphabetic cancel code is URL followed by *c.

- Example -
If registered code is http://tny.cz/abcd , then the alphabetic support, oppose, cancel SMS-codes are http://tny.cz/abcd , http://tny.cz/abcd*n , http://tny.cz/abcd*c respectively.

- Click on the numerical counts for the Support and Oppose totals for an issue to see all their detailed results with last 5 digits of mobile number giving their opinion for a URL.

- For an issue, the last sent SMS from a registered or activated mobile will replace the previous sent SMS for that issue. This makes the procedure immune to money buyout, gangster power and media power.
