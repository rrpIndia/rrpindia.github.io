- A new URL can be created only from a registered mobile. A newly created URL will be partially hidden. Full URL will be shown only if 25 or more registered mobiles send SMS-es to support it.
- Please see procedure of activation/registration at http://smstoneta.com/procedure-sms.php.

- After activation or registration of your mobile, you can open each URL and decide if you would like to support or oppose it. Numerical codes for supporting, opposing a URL or marking it as spam or unspam or cancelling your opinion is given in the table below.
- Only registered mobiles can mark a URL on the site as spam or unspam.
- Alphabetic Codes for supporting a registered URL is the URL itself to be sent to our server number, 08141277555 via sms.
- Alphabetic SMS-oppose code for a registered URL is the URL followed by *n , alphabetic cancel code, alphabetic spam code and alphabetic unspam code are URL followed by *c , *s and *u respectively.

- Example -
If registered URL is http://tny.cz/abcd , then the alphabetic support, oppose, cancel, spam and unspam SMS-codes are http://tny.cz/abcd , http://tny.cz/abcd*n , http://tny.cz/abcd*c , http://tny.cz/abcd*s , http://tny.cz/abcd*u respectively.

- Click on the numerical counts for the Support and Oppose totals for an issue to see all their detailed results with voter id, state and constituency of the registered persons giving their opinion for a URL.

- For a URL registered on the site, the last sent SMS from a registered or activated mobile will replace the previous sent SMS for that URL. This makes the procedure immune to money buyout, gangster power and media power.
